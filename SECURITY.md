# Sicherheitsrichtlinien

## API-Key Sicherheit

Dein Gold API Key ist vertraulich und sollte:
- ✅ Nur in Home Assistant konfiguriert werden
- ✅ Niemals in Git committed werden
- ✅ Regelmäßig rotiert werden (alle 6 Monate)
- ❌ Nie in Dokumentation oder Beispielen auftauchen

## Einen Sicherheitsvorfall melden

Falls du eine Sicherheitslücke findest, erstelle bitte ein Issue auf GitHub.
